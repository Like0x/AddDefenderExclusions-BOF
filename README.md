# AddDefenderExclusions-BOF
AddDefenderExclusions Beacon Object File

```
make -f Makefile.ADD
```
load cna

```
AddDefenderExclusions 1 C:\
```
You must supply an argument between 1 and 3.

1: Folder exclusions

2: Process exclusions

3: Extension exclusions.
